# Sticker Board

Generate image board using .NET and WPF UI.
This application can generate image board like the example below.

![image board example](https://github.com/Eryux/sticker-board/raw/main/Images/result.png)
**Important: Artworks on this image are copyrighted on made by the artist [Erixalu](https://twitter.com/ErixaluArt)**

### Download

You can download this application in [Release section](https://github.com/Eryux/sticker-board/releases) (Windows only)
*Note: .NET Framework 4.6.1 will be required to use the application.*


### Usage

![sticker board](https://github.com/Eryux/sticker-board/raw/main/Images/stickerboard.JPG)

* Download and unzip latest build from release section

* Launch the .exe

* Add your images by clicking on the "+"

* Set your preferences for colors, margins and spacing

* Click on generate and save the result image on your disk


### Build

#### Requirements

* .NET Framework 4.6.1
* Visual Studion 2019 or newer


#### Build

* Clone the git repository

* Open the .sln file with Visual Studio

* Right-click on StickerBoard solution then click on generate
